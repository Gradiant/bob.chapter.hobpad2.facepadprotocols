==================================================================
Challenges of Face Presentation Attack Detection in Real Scenarios
==================================================================
Chapter published in Handbook of Biometric Anti-Spoofing: Presentation Attack Detection, 2nd Edition.

If you use this package, please cite the following paper::

    @inproceedings{Costa-PazoHOBPAD2,
        author = {Artur Costa-Pazo, Esteban Vazquez-Fernandez, Jos\'e Luis Alba-Castro, Daniel Gonz\'alez-J\'imenez},
        title = {Challenges of Face Presentation Attack Detection in Real Scenarios},
        booktitle = {Handbook of Biometric Anti-Spoofing: Presentation Attack Detection, 2nd Edition.},
        year = {2018},
	month = feb,
    }

This package contains scripts to reproduce the results from the paper. The package also provides score files for IQM and GRADIANT PAD algorithms, the systems reported in the chapter.
